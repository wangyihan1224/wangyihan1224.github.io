纯水法&水辅法


        纯水法=> 不学辅技的水辅
        WIS < 30 (点一些WIS是为了SP量,有点出入梦就该叫法辅了)
        INT > 85
        INT > CON,AGI

        ---------------------

        纯水辅=> 不学攻击技能的水辅
        WIS > 53
        INT < 65-85 (很多纯水辅有点INT,狂雷或水雷出来前就归类为纯水辅吧)
        INT < AGI或CON

        ---------------------

        水辅法=> 较重视辅助的水法

        WIS > 53
        INT > 85
        INT < AGI或CON

        ---------------------

        水法辅=> 较重视攻击性水法

        WIS > 53
        INT > 85
        INT > AGI跟CON

        ==========================================

        目前版本

        纯水辅-属于极端型,转职开放前的团P主流

        水法辅-打任务主流,辅助能力非常好,又能帮打

        水辅法-打团P主流,速度快或拥有高血量.辅技又准

        纯水法-我还没看过有高级水法没学辅技的

        水战辅-攻击力,辅助能力都比水法辅"弱很多"

        水辅战-类似纯水辅,多了"一点点"攻击力

        纯水战-很类似火战,只是攻击力跟火战比"输超多"

        双修-战技法技都有学(恶搞用型)

        三修-战技,法技,辅技都学(也是恶搞用)

        ====================================================

        没见过有所谓的"纯"水法-水系不学辅助太可惜了

        就像地法也有学辅助-目前也没见过高级的纯地法

        风法大部分都点出圈圈或麻痹-纯风法也是没见过

        纯法师型--目前也只看过火法有人这样玩

        =====================================================

        "纯"通常都是战士系比较常见

        因为战系辅技效果不佳,可有可无

        法师学辅技效果很棒,几乎都有学

        学了辅技就不叫"纯"了..

        要改叫辅法或法辅

        在还不确定角色要注重辅助或偏重法师之前

        可以简称水法....

        但不是"纯"水法

        PS:

        别在把水"辅"叫成水"补"了...
