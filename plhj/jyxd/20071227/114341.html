职业配点-火杀手篇


        =基本能力=

        ATK=(等级*2+STR*2)*110%
        DEF=等级*2+CON*1.75
        MATK=等级*1.6+INT*2
        MDEF=等级*2+WIS*2.2
        SPD=等级*1.6+AGI*1.8
        隐藏加成：伤害值+约30%、被反击率微幅下降、初阶魔法威力微幅提升
        专属装备：杀手披风(+ATK10~80，等级越高能拿到的越好)

        火杀手，是目前一转职业中杀伤力最高的职业。火属性的攻击加成，配上杀手的职业加成，使这职业在任务中常担任打手的角色。以下乃针对此职业的各种配点方式做出分析。配点模式以转职100等为准。

        =力体型火杀手=

        STR 330↑
        CON 140~200
        INT 0
        WIS 18↓
        AGI 30↓
        此类型火杀手，算是较为基本的类型，同时拥有一定的血量与高杀伤力，较适合解任务时使用，SP也有一定的量，在解较小型任务时较没有吃补的困扰。解大型任务时则必须靠队友辅助，才能发挥全力。

        =力敏型火杀手=

        STR 320↑
        CON 110~150
        INT 0
        WIS 18↓
        AGI 50↑
        在敏捷上提升较多的火杀手，由于血量较少，较不适合解大型任务，但有较快的出手速度，要是使用得当，在单P或团P时都能有一定的威胁力。解小型任务时也有不错的表现。

        =力体型火辅杀手=

        STR 300↑
        CON 140~200
        INT 0
        WIS 42
        AGI 30↓
        算是力体型火杀手的变化型，由于中后期大型任务需要较高的持久力，而火战士最高杀伤力的技能"炽"也需要耗费大量SP，因此这类型的火杀手也开始变多。虽然基础攻击力和纯火杀有微幅差距，但相对的能提高一定续战力，中后期有不错的表现。

        =力敏型火辅杀手=

        STR 300↑
        CON 110~150
        INT 0
        WIS 42
        AGI 50↑
        力敏型火杀手的变化型。有高SP量，再加上不低的速度，再加上命中率极高的辅技"混乱术"，在PK场上占有不错的优势。解任务时虽然血量稍少了些，但由于SP量的关系，有不错的续战力。

        =纯力型火杀手=

        STR 400↑
        CON 150↓
        INT 0
        WIS 11↓
        AGI 10↓
        舍弃其它配点，专心追求高攻击力的类型。由于几乎不能先出手，在团P场上其实并不能有太大的优势，但若出手时出手攻击对方高等角色或放箭雨，则能造成极大伤害。这类型虽然有高攻击力，但极需要队友的辅助，否则无法发挥全力。若CON值压在110以下，则属于更极端的类型，虽有高杀伤力，但相对耐久力低，需要投资大量金钱购置高级装备才能维持一定程度的耐久力，而且大型任务时对队友辅助的依赖也会更高。

        =纯力型火辅杀手=

        STR 370↑
        CON 150↓
        INT 0
        WIS 18~42
        AGI 10↓
        纯力型火杀手的延伸版，为了避免后继无力的问题，牺牲了些许攻击力，换来一定的SP量。相较起来拥有较高的续战力，但速度方面还是须靠队友拉高平均速度。

        火杀手拥有极高的杀伤力，算是容易上手的职业。若是培养得当，在各方面都能成为相当的助力。原本克属的水系，也因为职业技"暗杀"的帮助，能打出不错的数字。希望在这篇中能让各位了解自己想要的培养方向，而培育出适合自己使用的火杀手。
