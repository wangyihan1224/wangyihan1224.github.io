初期宠物
　

        在飘流幻境中，初期因为任务因素而能加入玩家队伍中的宠物有四个！第一个是玩家一开始就会遇到的小猴子，其次则是在克兰村会加入玩家队伍的小兰跟萝卡，再来则是在北岛东北角海岸所遇到的少女妮丝。

        说到这边，或许有玩家会疑惑说，解完 克兰村山洞试炼任务之后， 萝卡不就会回到村长身边了吗？那么小编又是如何让萝卡加入队伍的呢？其实条件很简单，只要玩家完成 克兰村山洞试炼任务，等到萝卡回到村长身边之后，再去跟萝卡说话，萝卡就会加入玩家的队伍之中了喔！那么，小猴子、小兰、萝卡、妮丝，到底哪一只比较好呢？玩家又该练哪一只呢？小编就在这边就把这四个宠物的素质以及技能给列出来，让大家做个参考。

        从上面的能力列表来看，小猴子一开始的能力就已经偏低了！而小猴子的无敌头锤，虽然说要应付一开始的敌人是绰绰有余，但是当要应付 10 级以上的怪物时，已经显得不足了。所以小猴子只能说是官方为了让玩家一开始能有伴战斗，所送来应付一下的宠物而已。

        而小兰则是属于敏魔型！小兰的孤星只要其属性没有被克到，则随便都能打出 200 以上的素质 ( 即使有被克也可以打到 120 左右的伤害 ) 。不过小兰的缺点在于 HP 太少，不反则矣，一反就变成瞬间被自己的法术给秒杀。所以说，对于选小兰的玩家而言，得需控制小兰法术的威力！这时候或许会有玩家想到小兰的另一技能～石灵突。只可惜，此招只有对付水属怪物比较有用而已，面对其它属性怪物打起来的威力就感觉普普了。

        萝卡则是属于近战肉搏型，应该会有许多玩家蛮喜欢练这只的。萝卡主要常用的技能就是罗�x弯刀了。罗�x弯刀技能威力虽不如小兰的孤星来的猛，但是好歹平均威力也有 140 以上！而且萝卡血量略多，只要玩家的血量跟技能伤害有控制好，应该就比较不会挂掉了。

        妮丝则是法师型宠物，妮丝的技能威力也跟小兰的技能威力差不多猛！不过妮丝的缺点在于其技能太耗费 SP( 妮丝技能最少 60SP 起跳 ) 了，差不多打的两场就要进帐篷休息了。所以笔者感觉妮丝并不是说很实用，练起来也比较辛苦。

        所以玩家可以把宠物的栽培重点摆在小兰跟萝卡这两只人物上面。近战肉搏系的角色 ( 如席德、丹尼尔等等 ) ，可以考虑选择小兰作为搭配；法师辅助系的角色 ( 如洛克、 玛莉亚等等 ) ，可以考虑选择萝卡作为搭配。而懒得选择手动的玩家，选择近战肉搏系的角色 ( 如席德、丹尼尔等等 ) ，配上近战攻击强悍的萝卡，应该就可以用遥控器挂到天荒地老了吧！
